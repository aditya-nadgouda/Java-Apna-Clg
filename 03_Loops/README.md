# **Loops** in java

1. For 
2. While 
3. Do while 

---------------------------

## FOR :

**Syntax** :
`for (initialization ; condition ; updation){
// stm to execute
}
`
EX.: 
`for (int i = 0 ; i <= 100 ; i++){
sout ("Hello World ! ");
}`

## WHILE
**Syntax** :
`while (condition){
//stm to execute
}`

EX.: 
`int i = 0;
while (i<11){
sout (i);
i++;
}`

## DO-WHILE
**Syntax** :
`do{
//stm to execute
}while(condition);`

EX.:
`int i = 0;
do{
sout (i);
i++;
}while(i<11);`
