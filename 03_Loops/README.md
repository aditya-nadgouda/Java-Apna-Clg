# **Loops** in java

1. For 
2. While 
3. Do while 

---------------------------

## FOR :

**Syntax** :
<pre>
`for (initialization ; condition ; updation){
// stm to execute
}
`</pre>

EX.: 
<pre>
`for (int i = 0 ; i <= 100 ; i++){
sout ("Hello World ! ");
}`
</pre>

## WHILE
**Syntax** :
<pre>
`while (condition){
//stm to execute
}`
</pre>

EX.: 
<pre>
`int i = 0;
while (i<11){
sout (i);
i++;
}`
</pre>

## DO-WHILE
**Syntax** :
<pre>
`do{
//stm to execute
}while(condition);`
</pre>
EX.:
<pre> 
```int i = 0;
do{
sout (i);
i++;
}while(i<11);```
</pre>
